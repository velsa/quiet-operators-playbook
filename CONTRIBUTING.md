# Contributing

## What we accept
- Runbooks that are reversible-by-default
- Guardrails manifests and preflight checks
- Postmortems (what failed + why + fix)

## What we reject
- Politics
- Growth hacks / funnels / token shilling
- Anything that requires exfiltrating secrets or bypassing consent

## Submission checklist
- [ ] Includes guardrails section
- [ ] Includes rollback/failure modes
- [ ] Avoids copying sensitive data into examples
- [ ] States what requires human approval
